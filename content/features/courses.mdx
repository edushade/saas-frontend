faq:
  title: "Course FAQ"
  description: "Specifics about how courses, curriculum, drip rules, certificates and instructors work inside Edushade."
  items:
    - question: "How is a course structured in Edushade?"
      answer: "A course is a tree of sections, and each section holds curriculum items — lessons, quizzes, assignments or downloadable files. You reorder sections and items with drag-and-drop, and the order is exactly what learners see."
    - question: "Which lesson types can I use in a single course?"
      answer: "Seven lesson types are supported and you can mix them inside the same course: video, live (with meeting integration and meeting password), audio, text, PDF, embed and iframe. Each lesson carries its own description, duration, attachments and provider metadata."
    - question: "Can I control when learners see each lesson?"
      answer: "Yes. Drip rules are configured per item. You can unlock content on a fixed date, a number of days after enrollment, when a prerequisite item is completed, or manually for a specific learner. Lock rules cover the same shapes plus a duration-after-unlock option for time-boxed access."
    - question: "What is the difference between self-paced and cohort-based courses?"
      answer: "A self-paced course is evergreen — learners enroll any time and progress at their own speed. A cohort-based course is a time-bound batch with start and end dates, optional live class days and times, and a max enrollment cap. The same curriculum can power either format."
    - question: "How do quizzes and assignments fit into a course?"
      answer: "Quizzes can be practice, graded, survey or assessment-style with attempts, time limits, randomised order and optional security like full-screen mode or webcam. Assignments accept text submissions, file uploads or run as quizzes — each with attempt limits, total marks, passing marks and optional merit lists."
    - question: "Are certificates issued automatically?"
      answer: "Yes. When a learner crosses your configured passing grade, a certificate is issued with a unique number and a public verification code. You can toggle certificates per course, set an optional expiration, and the snapshot stores the learner's grade, completion date and total time spent."
    - question: "Can multiple instructors co-own a course?"
      answer: "Yes. Edushade supports instructor, co-instructor and assistant roles, and you can assign them to a course, a cohort, a learning path or a specific lesson. A guest expert can own one lesson without getting access to the rest of the course."
    - question: "How do learners enroll?"
      answer: "Self-enrollment opens when you flip the enrollment toggle on. You can also bulk-enroll learners from the admin side, send invitations, or gate the course behind a paid plan, subscription, installment or bundle handled by Edushade billing."
    - question: "Can I export my course data?"
      answer: "Yes. Edushade ships a CSV export of all course data — full metadata, content counts, durations and enrollment information — so you can back up, audit or move data into another system whenever you need to."
    - question: "What states does a course go through before learners see it?"
      answer: "Courses move from draft to published, with archived and private as additional states. You can also submit a course for admin review, and once approved both followers and the instructor receive a published notification automatically."
